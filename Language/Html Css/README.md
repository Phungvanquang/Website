# Hyper Text Markup Languag (HTML)

1. HTML (Hypertext Markup Language)
-  HTML (Hypertext Markup Language) Berners-Lee đã phát minh ra nó vào năm 1991. Ngày nay, HTML5 là phiên bản tiêu chuẩn và được tất cả các trình duyệt web hiện đại hỗ trợ. Tài liệu hướng dẫn HTML của chúng tôi cung cấp danh sách đầy đủ tất cả các phần tử HTML, bao gồm mô tả, ví dụ mã và bản xem trước trực tiếp. Chỉ cần cuộn xuống để duyệt tất cả các thẻ HTML theo thứ tự bảng chữ cái hoặc duyệt các thẻ theo danh mục của chúng.
