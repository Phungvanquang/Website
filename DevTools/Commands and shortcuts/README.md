[2. Run Command]()
- Chụp ảnh màn hình:
  + ```screenshot```: Chụp ảnh màn hình toàn bộ trang web.
  + ```capture node screenshot```: Chụp ảnh màn hình của một phần tử DOM đã chọn.
  + ```capture full size screenshot```: Chụp ảnh màn hình toàn bộ trang web, bao gồm cả nội dung nằm ngoài khung hình hiển thị.
- Hiệu suất và gỡ lỗi:
  + ```coverage```: Hiển thị phần trăm mã CSS và JavaScript đã được sử dụng trên trang.
  + ```performance```: Mở bảng điều khiển Hiệu suất để ghi lại và phân tích hiệu suất tải trang.
  + ```disable javascript```: Tắt JavaScript trên trang web hiện tại để kiểm tra khả năng dự phòng hoặc gỡ lỗi.
  + ```network conditions```: Cho phép bạn mô phỏng các điều kiện mạng khác nhau, chẳng hạn như kết nối chậm hoặc ngoại tuyến.
  + ```rendering```: Cho phép bạn mô phỏng các tính năng hiển thị CSS khác nhau.
  + ```show rendering```: Hiển thị bảng điều khiển Rendering.
  + ```clear console```: Xóa các thông báo trong bảng điều khiển Console.
- Phần tử và DOM:
  + ```inspect element```: Chọn một phần tử trên trang web để kiểm tra trong bảng điều khiển Elements.
  + ```show elements```: Hiển thị bảng điều khiển Elements.
- Mạng:
  + ```show network```: Hiển thị bảng điều khiển Network.
-Các chức năng khác:
  + ```show sources```: Hiển thị bảng điều khiển Sources.
  + ```show application```: Hiển thị bảng điều khiển Application.
  + ```show security```: Hiển thị bảng điều khiển Security.
  + ```show recorder```: Hiển thị bảng điều khiển Recorder.
  + ```settings```: Mở cài đặt DevTools.
  + ```device mode```: Bật hoặc tắt chế độ thiết bị để mô phỏng các thiết bị di động.

[4. Run snippet]()
- "Run snippet" (Chạy đoạn mã) trong Chrome DevTools là một tính năng cho phép bạn thực thi các đoạn mã JavaScript nhỏ trực tiếp trên trang web hiện tại. Điều này rất hữu ích cho việc kiểm tra, gỡ lỗi và tự động hóa các tác vụ.

[5. Go to symbol]()
- "Go to symbol" (Đi đến ký hiệu) là một chức năng rất hữu ích trong Chrome DevTools, đặc biệt khi bạn làm việc với các tệp JavaScript lớn. Nó cho phép bạn nhanh chóng tìm kiếm và điều hướng đến các hàm, biến và các ký hiệu khác trong mã nguồn của bạn.

[6. Go to line]()
Go to line" (Đi đến dòng) là một tính năng hữu ích trong Chrome DevTools, đặc biệt khi bạn muốn nhanh chóng di chuyển đến một dòng cụ thể trong tệp mã nguồn. Dưới đây là cách sử dụng "Go to line". Chỉ hiệu quả khi dùng trong tab source trong devtools, để tìm dòng mà file dang mở
