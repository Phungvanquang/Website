# Docker CLI


#### 2.1. xem image.
  ``docker image ls``
#### 2.2. xem container.
``  docker ps -a``
#### 2.3. chạy từ docker file lên thành image.
  ``docker build -t <name> ``
#### 2.4. chạy từ image lên container.
  ``docker run -p <port>:<port> -d <image>``
#### 2.5. xoá container.
 `` docker rm <container_id>``
